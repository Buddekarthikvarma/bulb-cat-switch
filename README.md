# Bulb and Cat Switch Application

A simple JavaScript DOM manipulation project that simulates turning a bulb on and off.

## Features

- Turn the bulb ON and OFF
- Change cat image based on bulb state
- Update switch status text dynamically
- Change button colors based on current state
- Built using HTML, CSS, and JavaScript

## Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## How It Works

### ON State
- Bulb glows
- Cat image is visible
- Status shows "Switched On"

### OFF State
- Bulb turns off
- Cat eyes glow in the dark
- Status shows "Switched Off"

## Learning Concepts

- DOM Manipulation
- Event Handling
- Changing Image Sources
- Updating Text Content
- Dynamic Styling with JavaScript
